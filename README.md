# News Application

A responsive **React-based news application** that fetches and displays real-time headlines from various categories and sources using a public News API.  
Designed with a clean, user-friendly interface to keep users updated with the latest news globally.



---

##  Features

-  **Real-Time News** — Fetches live headlines using a News API.  
-  **Category Filtering** — View news by topics such as Technology, Sports, Business, Health, and more.  
-  **Search Functionality** — Find specific articles or keywords.  
-  **Dark/Light Mode** — Toggle between themes for better readability.  
-  **Responsive Design** — Works seamlessly across all devices.  

---

##  Tech Stack

| Category | Technologies |
|-----------|---------------|
| **Frontend** | React.js, JavaScript (ES6+), JSX |
| **Styling** | Bootstrap / Tailwind CSS |
| **API** | NewsAPI.org (or similar) |
| **Deployment** | Vercel / Netlify |
